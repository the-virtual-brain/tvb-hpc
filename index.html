tvb-hpc¶

TVB-HPC is a package for generating high-performance code for TVB style simulations (neuroimaging-based anatomy and connectivity & neural mass models).

Currently effort is toward baseline code generation to be adapted and optimized with OpenMP or OpenACC.

Developers should refer the GitHub page.
